# About

It's an application to connect people who would love to help others to Non-governmental Organizations (NGOs).

<img src="docs/img/be-the-hero.gif" alt="Be the Hero Application" style="width: 100%; max-width: 600px;">

## Development

### Prerequisites

* Install [Node.js](https://nodejs.org/en/)
* Install [npm](https://www.npmjs.com/)

### Clone the repository

```sh
git clone https://github.com/ederchristian/be-the-hero.git && cd be-the-hero
```

## Installation

For each directory (backend, frontend and mobile), you may run:

```sh
npm install
```

Or:

```sh
yarn
```

## Usage

Running de project:

### Back-end, Front-end and Mobile

Using npm:

```sh
npm start
```

Or:

```sh
yarn start
```

## Tests

Using Jest:

```sh
cd backend && npm test
```

Or:

```sh
cd backend && yarn test
```
